Frequently Asked Questions

How to Set it up?

nce you get access to the account, you can create a form internally, it will generate a javascript link for you all you have to do is paste the javascript link into your website and its ready to capture all the details.

Do i need to hire a dev to do i?

Nope, you can do it yourself, all you have to do is paste the javascript link into your website or ask your developer to do so. It needs no additional experience to do so.

Does it work on Wordpress or other no-code tools?

Yes it does, its designed to work via a javascript to identify any form and capture its data. For wordpress you can add it via a snippet plugin. Same goes for other no code tools.

Can i use it for multiple pages/forms?

Yes, thats the special feature of Rabbitforms, you can integrate it in multiple pages/forms and get all data in one place.
